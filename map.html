헷갈리는 3가지 (여기서 많이 틀림)

통계 문제 → 일반·연도별 = S1 · 용도별 = S20 · 원인별 = S3
피난 문제 → 용도가 명시되면 S20 · 없으면 S4
Fire stop → 정의·3종류 = S18 Ch1 · 등급·사이클 = S18 Ch5
Wildland 지붕 → S1 pg86 (S19 아님!)

문제 하나 = 4단계

1. 키워드 하나 짚기 (K-factor / ESFR / hanger / standpipe)
2. 섹션 → 위 표
3. 챕터 → 탭 또는 색인
4. 헤드라인만 10~15초 훑기 → 표에서 답 찾기 (계산하지 말 것)
한 문제 3분 넘기면 찍고 넘어간다. 25개 틀려도 합격.

권 1 = S1–S11 (회사) · 권 2 = S12–S20 (집) · ★★★ = 문제은행 35문+
